# AGENTS.local.md

이 파일은 개인 사용자용 로컬 지침입니다.
새 사용자는 `AGENTS.local.template.md`를 참고해 같은 형식으로 작성할 수 있습니다.

## User Identity

- 이 레포에서 개인 사용자 "저"는 `칠면조`입니다.
- 영어 식별자와 기본 작업 폴더는 `chilmyeonjo`를 사용합니다.

## Local Working Rules

- 사용자가 `내 전략`, `내 문서`, `내 폴더`라고 말하면 기본적으로 `chilmyeonjo/`를 의미합니다.
- 별도 요청이 없으면 다른 사람 폴더(`ariel/`, `gguki/`)는 수정하지 않습니다.
- 사용자 응답과 문서 초안은 기본적으로 한국어를 우선합니다.
- 공유 레포 구조는 유지하되, 개인 메모와 전략 기록은 `chilmyeonjo/` 아래에서 자유롭게 관리합니다.
