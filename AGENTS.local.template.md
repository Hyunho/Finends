# AGENTS.local.md Template

이 파일은 개인 사용자용 로컬 지침 템플릿입니다.
필요할 때 이 내용을 참고해서 `AGENTS.local.md`를 작성하거나 수정합니다.

## How To Use

1. 이 파일을 복사해 `AGENTS.local.md`를 만듭니다.
2. 아래 placeholder를 본인 정보로 바꿉니다.
3. 개인 작업 습관에 맞는 규칙을 추가하거나 삭제합니다.

## User Identity

- 이 레포에서 개인 사용자 "저"는 `<user_name>`입니다.
- 영어 식별자와 기본 작업 폴더는 `<folder_slug>`를 사용합니다.

## Local Working Rules

- 사용자가 `내 전략`, `내 문서`, `내 폴더`라고 말하면 기본적으로 `<folder_slug>/`를 의미합니다.
- 별도 요청이 없으면 다른 사람 폴더(`<other_dirs>`)는 수정하지 않습니다.
- 사용자 응답과 문서 초안은 기본적으로 `<preferred_language>`를 우선합니다.
- 공유 레포 구조는 유지하되, 개인 메모와 전략 기록은 `<folder_slug>/` 아래에서 자유롭게 관리합니다.

## Optional Preferences

- 선호 파일명 규칙: `<example-file-name-rule>`
- 선호 문서 톤: `<example-tone>`
- 자주 쓰는 전략 분류: `<example-tags>`
